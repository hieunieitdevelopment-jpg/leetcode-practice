# 🧩 Các Pattern phổ biến trong LeetCode

## 1. HashMap / Hash Set
- **Khi nào dùng:** Tìm cặp, đếm tần suất, kiểm tra tồn tại
- **Bài tiêu biểu:** Two Sum, Contains Duplicate, Group Anagrams
- **Trick:** complement = target - current

## 2. Two Pointers
- **Khi nào dùng:** Mảng đã sort, tìm cặp, palindrome
- **Bài tiêu biểu:** Two Sum II, Valid Palindrome, Container With Most Water

## 3. Sliding Window
- **Khi nào dùng:** Substring/subarray liên tục, max/min trong window
- **Bài tiêu biểu:** Maximum Subarray, Longest Substring Without Repeating

## 4. Stack
- **Khi nào dùng:** Matching brackets, monotonic stack
- **Bài tiêu biểu:** Valid Parentheses, Daily Temperatures

## 5. Binary Search
- **Khi nào dùng:** Mảng sorted, tìm vị trí, tối ưu O(log n)
- **Bài tiêu biểu:** Binary Search, Search Insert Position

## 6. BFS / DFS
- **Khi nào dùng:** Tree, Graph traversal
- **Bài tiêu biểu:** Max Depth of Binary Tree, Number of Islands

## 7. Dynamic Programming
- **Khi nào dùng:** Tối ưu, đếm cách, có overlapping subproblems
- **Bài tiêu biểu:** Climbing Stairs, House Robber, Coin Change

## 8. Linked List
- **Khi nào dùng:** Reverse, detect cycle, merge
- **Bài tiêu biểu:** Reverse Linked List, Merge Two Sorted Lists, Linked List Cycle
